# 🏢 Employee Attrition Intelligence System

An end-to-end Machine Learning project that predicts employee attrition using HR analytics and provides actionable insights through an interactive Streamlit dashboard.

---

## 📌 Project Overview

Employee attrition is a major challenge for organizations, leading to increased hiring costs and reduced productivity. This project predicts whether an employee is likely to leave the company based on various HR-related factors using Machine Learning.

The application also provides:

* 📊 Interactive HR Dashboard
* 🎯 Employee Attrition Prediction
* 📈 Prediction Confidence Score
* 💡 Personalized Retention Suggestions
* 🌙 Modern Streamlit User Interface

---

## 🚀 Features

* Employee Attrition Prediction
* Interactive Dashboard with HR Analytics
* Prediction Confidence Score
* Personalized Retention Suggestions
* Department-wise Attrition Analysis
* Employee Statistics Visualization

---

## 🤖 Machine Learning Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* K-Nearest Neighbors (KNN)
* Support Vector Machine (SVM)
* Gaussian Naive Bayes

The best-performing model was selected for deployment.
